1. Open Github
  2. click on new
  3. name your repo
  4. make it public
  5. check the readme box
  6. everything else on this page is fine as is
  7. click on create repository
  8. click on code and make sure ssh is highlighted
  9. click on two overlapping squares
  10. open up your terminal
  11. choose a location for your repo and cd into it step by step by utilizing ls and then cd'ing into whichever folder you want to (desktop with a folder is fine)
  12. type git clone and then paste the stuff from the two overlapping squares
  13. cd into your repo
  14. create your files either using the terminal or using vscode

for the terminal

cd is to change folder

ls lists all folders you can cd into currently

touch allows you to create a file

mkdir allows to create a folder

cd .. goes back a folder
